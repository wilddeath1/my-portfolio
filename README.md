# 🌐 Personal Portfolio Website

This repository contains the source code for my personal portfolio website.  
The website showcases my projects, skills, and creative work in a clean, modern, and responsive layout.

It is designed to provide a smooth browsing experience across different screen sizes and devices.

---

## 🚀 Live Website

🔗 **View Live Portfolio:**  
https://ayush-tyagi-gold.vercel.app/

---

## 📌 Features

- Responsive design for desktop, tablet, and mobile
- Modern and minimal user interface
- Smooth scrolling navigation
- Project showcase section
- Resume download functionality
- Optimized images and assets
- Clean and organized file structure
- Fast loading performance

---

## 🛠️ Technologies Used

- **HTML5** — Page structure
- **CSS3** — Styling and layout
- **JavaScript (Vanilla JS)** — Interactivity
- **GitHub** — Version control
- **Vercel** — Deployment and hosting
